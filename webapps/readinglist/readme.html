JAFER Client - RC3

This is an initial release of the JAFER client for public review. Documentation is currently incomplete.

Contents of Archive

jafer-client.jar
Library for using JAFER client in Java applications
src.jar
Source for JAFER client
doc.jar
JavaDoc for JAFER client
JAFER.war
Sample web application using JAFER client

Sample application

Simple web application demonstrating searching and handle the result records either programmatically or via XML.

To use install JDK1.4 and TomCat 3.3. Place the JAFER.war file into the TomCat webapps folder. War archive should expand automatically when TomCat is started.
