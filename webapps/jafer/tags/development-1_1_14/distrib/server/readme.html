JAFER Server

This is a development version of the JAFER software. Documentation is currently incomplete.

Contents of Archive

org folder
JAFER Java Classes and resources.
lib folder
Library jars used by JAFER code.
jaferdoc.jar
Javadoc archive.
ServerStart.bat
Starts server (see org.jafer.test.ServerStart in Javadoc for details).
ServerTest.bat
Starts server, and does a test search (see org.jafer.test.ServerTest in Javadoc for details).
TestClient.bat
Runs a test search on a running server (see org.jafer.test.TestClient in Javadoc for details).

Configuration:

A Servlet based admin interface is being developed, but in the meantime, the server can be configured by editing the server.xml file in the org.jafer.conf package before starting the server.
