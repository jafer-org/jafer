JAFER Client

This is an initial release of the JAFER client for public review. Documentation is currently incomplete.

Contents of Archive

org folder
JAFER client Java Classes
lib folder
Library jars needed by JAFER
jaferdoc.jar
Javadoc for JAFER
Demo.bat
Windows batch file to run test searches.

Running a test search

Demo.bat can be used to run a test search. The test can be modified by editing Demo.java in org/jafer/test, and recompiling.

Sample JSP (Java Server Pages) application

(This is available as a separate download, see www.jafer.org)

Simple web application demonstrating searching and handle the result records either programmatically or via XML.

To use install JDK1.4 and Apache Tomcat 3.x or 4.x. Place the JaferSample.war file into the Tomcat webapps folder. War archive should expand automatically when Tomcat is started.

Search interface available at: http://localhost:8080/JaferSample/query.html
